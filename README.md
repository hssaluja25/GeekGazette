## Getting Started

1. `git clone git@github.com:hssaluja25/hackernews.git`
2. `cd hackernews`
3. `flutter pub get`
4. `flutter run`

## Features

* Uses Flutter 3.0
* Displays top stories on Hacker News
* Supports pull down to refresh
